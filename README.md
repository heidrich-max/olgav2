# Olgav2

Das neue Repository für die Weiterentwicklung des Olga-Projekts.

## Funktionen
- **Auto-Deployment**: Jeder Push in den `main` Branch wird automatisch auf `dev.frankgroup.net` hochgeladen.
- **Synchronisiert**: Ermöglicht die Arbeit auf mehreren Geräten.

## Entwicklung
1. Änderungen lokal vornehmen.
2. `git add .`
3. `git commit -m "Deine Beschreibung"`
4. `git push origin main`

Die Änderungen sind kurz darauf live auf dem Webserver zu sehen.
