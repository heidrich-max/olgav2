# Projekt-Regeln & Arbeitsbedingungen (Olgav2)

Diese Datei enthält spezifische Vorgaben für die Arbeit an diesem Projekt. Bitte beachte diese Regeln bei jeder Interaktion.

## Kommunikation
- **Sprache**: Antworte immer auf **Deutsch**.

## Datenbank-Zugriff
- **Host**: `cms.frankgroup.net`
- **Modus**: **Nur Lesezugriff (Read-Only)**.
- **Wichtig**: Bevor Änderungen an der Datenbank vorgenommen werden (Insert/Update/Delete), muss eine explizite Bestätigung vom Benutzer eingeholt werden.

## Deployment & Workflow
- **Live-System**: Das Projekt ist unter `dev.frankgroup.net` erreichbar.
- **Git**: Jede Code-Änderung muss committed und per `push` an GitHub übertragen werden. **Ein `git pull` auf dem Server erfolgt automatisch nach dem Push**, sodass Änderungen sofort aktiv werden.
